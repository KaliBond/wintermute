## USAGE GUIDE

This template is designed for **narrative diagnostic reports** on any society's institutional health across the 8 CAMS nodes. It balances:
- **Quantitative rigor** (metrics, tables, historical phase portraits)
- **Structural systems analysis** (pathological signatures, risk pathways)
- **Narrative depth** (what the society feels like, civilisational meaning)

The template assumes:
1. A dataset with **8 nodes** (Helm, Shield, Archive, Lore, Stewards, Craft, Hands, Flow) scored across **4 dimensions** (Coherence, Capacity, Stress, Abstraction)
2. **Temporal depth** (at least 50+ years, ideally 100+) to construct historical phase portraits
3. **Derived metrics** (Node Value, Bond Strength, Cognition, Affect) computed from base scores
4. **System scalars** (M(t), Y(t), Headroom, Θ) that characterise the entire system state

## TECHNICAL NOTES FOR REPORT WRITERS

### On Pathological Archetypes
Common diagnostic patterns in CAMS datasets:

1. **Executive Decoupling**: High Helm Node Value but very low Bond Strength to other nodes (Germany 2025)
2. **Institutional Fragmentation**: Archive strong but Helm weak; slow-loop dominates fast-loop (France 2026)
3. **Hands Erosion**: Lowest Node Value + highest Stress; labour systems failing (Australia, Germany)
4. **Mythic Collapse**: Y(t) crashes below historical mean; narrative systems fracturing (USA 2025)
5. **Stewards Dominance**: Single node drives overall system resilience; system is structurally unbalanced (Germany, France)
6. **Reactive Dominance**: Θ > 1.0; system operating in crisis mode; slow-loop institutions cannot coordinate
7. **Headroom Deficit**: Capacity ≤ Stress; system running on reserve; shock-absorption capacity exhausted

### On Choosing Historical Comparison Points
- Use **50–100 year spans** to identify secular trends
- **Anchor on known crises** (wars, revolutions, economic collapses) to calibrate stress levels
- **Mark inflection points** where trajectory visibly changes
- **Note regime transitions** (constitutional, economic, geopolitical)
- **Identify cyclical patterns** (e.g., does this society show recurring 20–30 year cycles?)

### On Narrative Voice
This template supports a **literary-analytical tone** that integrates quantitative diagnosis with civilisational meaning. The goal is not academic objectivity but **honest phenomenology** — what does the CAMS data suggest about lived experience?

- Use **specific language** for emotions/states (e.g., "exhaustion" not "stress," "betrayal" not "dissatisfaction")
- **Name the historical analogues** when patterns repeat
- **Avoid false neutrality** — the data shows what it shows; interpretation is honest synthesis, not bias
- **Embed the metrics** in prose rather than detaching them into tables alone
- **Conclude with clarity** about what is at stake (what outcome does this lead to if uncorrected?)
