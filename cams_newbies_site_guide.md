## 2. Core Concepts — What CAMS Actually Is

- **The CAMS Model** — https://neuralnations.org/model
  - The **Eight Nodes** (Helm, Shield, Lore, Archive, Hands, Craft, Flow, Stewards)
  - The **Four Metrics** per node (C, K, S, A)
  - Core formulas:
    - Node Value: `V_i = C_i + K_i - S_i + 0.5*A_i`
    - System Value: `V_bar` (mean) and dispersion `σ_V`
    - Bond Strength and system-wide coordination `Λ(t)`
  - Phase space regimes (Coherent-Capable, Crisis/Transition, Coordination Failure)
  - Current version notes (v3.2-R, CAM5 ensemble, kappa thresholds, attractors)

- **Key Ideas to Internalise**
  - Societies as **coordination systems**, not just collections of people or GDP
  - Institutions as "exteriorised sub-brains"
  - Retrodiction vs prediction
  - Blinded ensemble scoring methodology
  - Galilean (observation-driven) rather than purely theoretical approach

## 6. How to Use This Site — Recommended Paths

**Path A — Casual Explorer (30–60 mins)**
1. Start Here → FAQ
2. Model page (focus on eight nodes diagram)

**Path B — Serious Newbie (2–4 hours)**
1. Start Here + FAQ
2. Full Model page + Bond Strength spec
3. Open Datasets (download one country you know)

1. Start Here (Practitioner section)
2. Open Live Dashboard
