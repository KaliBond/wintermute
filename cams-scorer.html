CAMS Scorer

Score any country, company, or city on the eight CAMS nodes, on demand. Five isolated GPT passes read the historical record, score Coherence, Capacity, Stress, and Abstraction independently, and are combined into an ensemble mean with an uncertainty envelope — the same raw-scoring doctrine behind every dataset published on this site.

Runs are capped server-side at 5 years and 5 passes per request, and rate-limited per visitor, to keep this a fair, shared instrument. Passes run one at a time — a 3-pass run typically takes 2–4 minutes, longer for more passes or years. Leave the tab open until it finishes.

How this differs from the published datasets

This is the same raw-scoring rubric used to build every ensemble dataset on Neural Nations, run live instead of offline. It captures raw node measurements only — no Node Value, Bond Strength, or diagnosis — leaving all downstream calculation to the published CAMS pipeline. Treat a single on-demand run as indicative, not canonical: the published datasets use larger, cross-checked ensembles.
