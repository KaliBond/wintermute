CAMS Scorer — Local Edition

A standalone Windows tool that runs the same CAMS raw-scoring ensemble as the online CAMS Scorer, entirely on your own machine, using your own OpenAI account. Nothing is sent to Neural Nations — the tool talks directly to OpenAI with your key, and you're billed directly by OpenAI at your account's own rates.

cams-scorer.exe
Windows · ~19 MB · no install, no Python required
Download
Windows SmartScreen will likely flag this on first run, since it's a new, unsigned executable from an independent publisher. Click "More info" → "Run anyway" if you trust the source. You can also read the plain-text README or inspect the source repo first.

Setup

  1. Get an OpenAI API key at platform.openai.com/api-keys.
  2. Open a terminal (PowerShell) in the folder where you saved cams-scorer.exe.
  3. Set your key for the session: $env:OPENAI_API_KEY = "sk-..."
  4. Run a scoring pass — see examples below.

Usage

# country, single year, default 5 passes
.\cams-scorer.exe country USA 2020 2024

# company
.\cams-scorer.exe company Tesla 2023

# city, fewer passes, faster
.\cams-scorer.exe city Detroit 2019 2023 --passes 3

Each pass is a separate live model call — a 5-pass run typically takes several minutes. Two CSVs are written to the current folder: <entity>_cams_scores.csv (ensemble mean per node) and <entity>_cams_envelope.csv (uncertainty — standard deviation and valid-pass count per cell). This is raw scoring only — no Node Value, Bond Strength, or diagnosis. See the model page for the full CAMS calculation pipeline.
